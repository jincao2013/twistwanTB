# twistwanTB
Ab initio four-band Wannier tight-binding model for generic twisted graphene systems.

### examples

- Magic angle twist bilayer graphene (mTBG) without h-BN substrate (theta=1.08 degree)
- Magic angle twist bilayer graphene (mTBG) with h-BN substrate (theta=1.08 degree)
- Twist double bilayer graphene (TDBG) (theta=2.0 degree) 
- Magic angle twist double bilayer graphene (mTDBG) with h-BN substrate and displacement field turned on (theta=1.248 degree) 
- Magic angle twist trilayer graphene (mTTG) with h-BN substrate (theta=1.248 degree) 
- Twist multilayer graphene  (theta=2.646 degree)
